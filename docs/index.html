Usage Guide

1

Include the cropo using ES modules

import cr from 'cropo';

Or include the cropo using CommonJs

const cr = require('cropo');

initialize Cropo with the following code

// load html canvas element
cr.loadCanvas(document.getElementById('canvas'));
// load image
cr.loadImageFromUrl('http://address/landscape/3.jpg');

2
3

Optionals

// select an input to control zoom level
cr.loadSlider(document.getElementById("myRange"))

input range most be between 1 and 5

Download or upload image

// download canvas image
cr.download();
// get image as data url
cr.getDataUrl();
